# 📝 To-Do List App

A simple, responsive, and interactive To-Do List application built using **HTML, CSS, and JavaScript**.  
This project demonstrates core front-end development concepts including DOM manipulation, event handling, state management, and data persistence using localStorage.

---

## 🚀 Features

- Create, read, update, and delete tasks (CRUD)
- Mark tasks as completed or active
- Filter tasks (All / Active / Completed)
- Persistent storage using `localStorage`
- Dynamic DOM updates with event delegation
- Responsive and clean user interface

---

## 🛠️ Tech Stack

- HTML5  
- CSS3  
- JavaScript (ES6)

---

## 🧠 Key Concepts Used

- DOM Manipulation  
- Event Handling  
- Local Storage API  
- State Management  
- Event Delegation
